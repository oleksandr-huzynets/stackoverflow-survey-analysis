# Stack Overflow Developer Survey Analysis

## Project Overview
This project presents exploratory data analysis (EDA) of the Stack Overflow Developer Survey dataset using Python.

## Objectives
- Analyze survey response distribution
- Calculate statistical metrics for work experience
- Evaluate remote work trends
- Analyze popularity of Python
- Conduct salary analysis by country
- Explore education levels of high-income developers

## Tools Used
- Python
- Pandas
- Data Analysis
- Statistical Aggregation

## Key Insights
- Identified distribution of developer experience levels
- Analyzed remote work adoption
- Evaluated salary trends among Python developers
- Compared country-level income differences

## File Included
- `stackoverflow_survey_analysis.ipynb` – full analysis notebook
